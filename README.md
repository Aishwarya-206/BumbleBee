# BumbleBee

A static website dedicated to bumblebees, built as a front-end practice project using HTML and CSS.  
The goal of this project is to combine clean structure, simple styling, and a soft nature-inspired theme.

---

## About

This website is a small personal project created to explore basic web development concepts while building something meaningful.  
It features informative content about bumblebees along with multiple pages, styled components, and visual assets to create a complete mini website.

---

- Welcome Page
